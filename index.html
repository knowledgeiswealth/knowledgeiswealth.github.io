💡 Practical Tips

🛣️ Narrow Roads (< 6m)

Keep vehicle centered. Both LiDARs have good coverage. Use 0° tilt.

🏢 Wide Roads (> 12m)

Drive closer to buildings. Use +5° to +10° tilt for upper floors.

🏙️ Tall Buildings

Use +10° to +15° tilt. Reduce speed for denser point clouds.

🚗 Optimal Speed

10-20 km/h for good coverage. Slower = more points per area.

📷 Why Camera Alone Cannot Measure Building Height

A single camera image cannot directly determine the real height of a building. Here's why:

❌ Limitations of Monocular Vision

1. No Depth Information

A 2D image loses all depth information. Two buildings of different heights at different distances can appear the same height in pixels.

2. Unknown Scale

Without a known reference object (like a person, car, or measured feature), there's no way to convert pixels to real-world meters.

Height (pixels) ≠ Height (meters) without scale factor

3. Perspective Distortion

Objects farther from the camera appear smaller. A 10m building at 5m distance looks taller than a 20m building at 30m distance.

4. Camera Parameters Unknown

Height estimation requires: focal length, sensor size, and camera tilt angle. These must be known or calibrated precisely.

✅ What Camera CAN Detect

1. Roofline Detection

Camera can reliably detect where the roof edge is in the image. Combined with LiDAR, this gives the actual roof height.

2. Relative Proportions

Camera can measure relative sizes (e.g., "window is 1/5th of building height"). If window height is known, building height can be estimated.

3. Floor Count

Camera can count visible floors and windows. If typical floor height is known (3-4m), estimated height ≈ floors × floor_height.

🔑 Key Insight: Sensor Fusion

LiDAR provides accurate distance → Camera provides roofline detection → Combined = accurate height

Hestimated = max(HLiDAR, Hcamera_roof)

📐 Height from Single Image (Requires Assumptions)

If you must estimate height from a single image, you need:

  • Known distance to building (from GPS or other sensor)
  • Camera calibration (focal length, sensor size)
  • Camera pose (tilt angle relative to horizontal)
  • Formula: Hreal = (Hpixels × distance) / f × (sensor_height / image_height)

⚠️ All assumptions must be accurate — small errors in distance or tilt cause large height errors.

📐 Building Height Estimation - LiDAR + Camera Fusion

When LiDAR doesn't fully cover a building (limited FOV, tilt angle, distance), camera data can help estimate the complete height.

Red dots = LiDAR points (partial coverage) Purple line = Camera-detected roofline

🔬 How It Works

Step 1: LiDAR Point Analysis

LiDAR provides accurate 3D coordinates for points it captures. The highest detected point gives a minimum building height.

HLiDAR = max(Zpoints) = partial height

Step 2: Camera Roofline Detection

Camera image analysis (edge detection, line detection) finds the building roofline even where LiDAR points are missing.

Yroofline = edge detection in image

Step 3: Fuse & Estimate

Project camera roofline to 3D space using extrinsic calibration and combine with LiDAR data.

Hestimated = max(HLiDAR, Hcamera_projected)

📊 Methods for Height Estimation

LiDAR Extrapolation

Use highest LiDAR points + building edge slope to extrapolate total height.

H = Zmax + ΔHextrapolate

Camera Projection

Detect roofline in image, project to 3D using camera extrinsics with ground plane.

H = (Yroof - Yground) × scale

Vanishing Point

Use building edge lines to find vanishing point, estimate height from perspective.

H = f × (vtop - vbottom) / distance
